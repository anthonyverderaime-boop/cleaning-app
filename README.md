## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Supabase setup for dashboard calendar

Run the SQL in `supabase/cleaning_assignments.sql` in your Supabase SQL editor to create the calendar assignment table and policies used by `/dashboard`.

## Supabase setup for admin-editable checklist

Run `supabase/checklist_admin.sql` in your Supabase SQL editor, then add the admin user to `app_admins`:

```sql
insert into public.app_admins (user_id)
values ('YOUR_AUTH_USER_UUID')
on conflict do nothing;
```

## Supabase setup for clean history

Run `supabase/cleaning_history.sql` in your Supabase SQL editor. This creates per-user daily completed-cleaning records used by `/dashboard/history`.
